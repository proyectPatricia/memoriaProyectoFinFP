⭐Proyecto Intermodular – Desarrollo de Aplicaciones Web (DAW)
Este repositorio contiene el proyecto intermodular del ciclo Desarrollo de Aplicaciones Web, donde se integran tecnologías de cliente, servidor y despliegue para construir una aplicación web completa y funcional.

📌 Módulo: Desarrollo Web en Entorno Cliente (DWEC)
Implementación de varias prácticas AJAX utilizando jQuery y JavaScript puro, consumiendo APIs externas en formato XML y JSON.
Incluye:

Peticiones AJAX con jQuery (XML y JSON).

Peticiones AJAX con XMLHttpRequest (XML y JSON).

Procesamiento dinámico de datos y actualización del DOM.

Uso de APIs como ECB y RandomUser.

📌 Módulo: Desarrollo Web en Entorno Servidor (DWES)
Desarrollo de un sistema de votación en tiempo real para productos:

Login con validación mediante AJAX.

Votación de productos (1–5 estrellas) con actualización dinámica.

Prevención de votos duplicados por usuario.

Cálculo de medias y representación gráfica de estrellas.

Arquitectura basada en DAO, PDO y BladeOne como motor de plantillas.

Base de datos MySQL con tablas de usuarios, productos y votos.

📌 Módulo: Despliegue de Aplicaciones Web (DAWDAW)
Configuración completa de un servidor DNS maestro con BIND en Linux:

Creación de zona DNS para dawdistancia.net.

Registros SOA, NS, A y CNAME.

Verificación con herramientas como dig.

Pruebas de resolución y funcionamiento del servicio.

🛠️ Tecnologías utilizadas
HTML5, CSS3, JavaScript, jQuery

PHP, PDO, BladeOne

MySQL

AJAX

BIND9 (DNS)

Composer

Ubuntu + VirtualBox

NetBeans IDE

Firefox Developer Tools

🎯 Objetivo del proyecto
Integrar las competencias del ciclo DAW mediante el desarrollo de una aplicación web completa, aplicando buenas prácticas de programación, arquitectura, seguridad y despliegue.
